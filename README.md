## Projetos em Destaque

### [Conectando e Doando](https://github.com/diogolopesnunes/conectando-e-doando)
Plataforma web desenvolvida para conectar ONGs e doadores, facilitando a divulgação de causas sociais e o recebimento de doações. O sistema conta com cadastro de usuários, diferenciação entre doadores, ONGs e administradores, autenticação, gerenciamento de perfis, aprovação de ONGs, integração com PIX e geração de QR Code para pagamentos.

A aplicação foi estruturada com frontend em React/Vite e backend em Python Flask, utilizando Firebird SQL como banco de dados. O repositório possui separação entre backend e frontend, e o frontend usa React, React Router, Google Charts, máscaras de input e SweetAlert2.

Resumo curto:
Sistema full stack para aproximar ONGs e doadores, com cadastro, autenticação, gerenciamento de perfis, integração PIX, QR Code, relatórios e dashboards.

### [Auro](https://github.com/RafaelMelin33/Auro)
Sistema web de gestão financeira e simulação de crédito voltado para pequenas empresas. A aplicação permite o controle de receitas, despesas, movimentações financeiras, saldo, usuários e taxas, além de oferecer dashboards para análise mensal e anual das finanças.

O projeto foi desenvolvido em Python com Flask, usando Firebird SQL como banco de dados, templates HTML, CSS e JavaScript. Também possui autenticação, perfis de usuário e administrador, cadastro de usuários, controle de acesso e geração de gráficos com Pygal para visualizar receitas e despesas ao longo do ano. O próprio repositório descreve o Auro como um “sistema de gestor e simulador de crédito para pequenas empresas”.

Resumo curto:
Sistema de gestão financeira para pequenas empresas, com controle de receitas, despesas, saldo, taxas, usuários, dashboard e simulação de crédito.
